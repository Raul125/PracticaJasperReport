Conexión a la Base de Datos

Esta opción permite conectar la aplicación a la base de datos para gestionar los datos de Naves, Tripulantes y Misiones.

Cómo Conectar a la Base de Datos

  1. Accede al menú de Preferencias.
  2. Selecciona la opción Conectar a Base de Datos.
  3. Introduce los parámetros de conexión requeridos:
    • Host
    • Puerto
    • Nombre de la Base de Datos
    • Usuario
    • Contraseña
  4. Haz clic en Conectar.

Si la conexión es exitosa, recibirás una notificación y podrás empezar a interactuar con la base de datos.
